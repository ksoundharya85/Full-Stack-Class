5. Document Structure Tags

This entire page demonstrates document structure:

  • <!DOCTYPE html> - Document type declaration (top of file)
  • <html> - Root element wrapping entire content
  • <head> - Contains meta information and title
  • <title> - Sets the browser tab title
  • <body> - Contains all visible content on the page
